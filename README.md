The "uvvis_analysis.py" file is a local python code for running the UV-Vis Similarity Calculator on your terminal. It is recommended to use as follow:

python uvvis_analysis.py -e experimental_data.dat -s tddft_transitions.stk -o output_name --plot-emin e_min --plot-emax e_max

- The experimental_data file must be a .csv or .dat file (energy in eV, intensities - whitout title rows);

- The tddft_transitions must be a .stk file generated using the orca_mapspc program;

- e_min and e_max should be given as numbers in eV;
  
- The output_name will be the prefix that the tables and images names generated by the code will have;

All files have to be on the same folder as the python code. The output for this code will be a pdf file containing all ploted data, a .csv file containing all calculated data and the print of the results in the terminal. The "experimental_data_example.dat" and "STK_file_example.stk" files can be used as a test for running the code.
